# CampusEvent: Student Club Management System

CampusEvent is a web-based system developed to manage university student club activities.

## User Roles

- Administrator
- Club Leader
- Student

## Main Features

### Administrator
- Manage programme proposals
- Approve or reject proposals
- Manage club events
- Manage registrations
- Generate reports
- Generate certificates
- Manage users

### Club Leader
- Submit programme proposals
- Upload posters and supporting documents
- Edit pending proposals
- Generate official proposal reports
- View approved events

### Student
- View approved events
- Register for events
- View registration status
- View and download certificates

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Installation

1. Copy the project folder into the XAMPP `htdocs` folder.
2. Import `student_club_system.sql` into phpMyAdmin.
3. Start Apache and MySQL.
4. Open:

   http://localhost/club_system/
